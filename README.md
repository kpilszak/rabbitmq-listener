## General info
This project is a Spring application.  
The main purpose is to create a demo listener for RabbitMQ.  

## Technologies
Project is created with:
* Java  
* Spring Boot 2
* Spring MVC
* RabbitMQ  
  
IDE: Intellij IDEA 2020
